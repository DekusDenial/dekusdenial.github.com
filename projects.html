The release version I built was in Windows, where I have used a gem called ocra to build Windows executable. You can try out the same thing by installing the gem: gem install ocra and then run ocra dbms.rb [... append remaining .rb files in the lib directory here without these embracing brackets]

UITableView Accordian

This was supposed to be a iOS utility applicaiton at first. However due to the unfortunate fact that someone stood up me on this project. I decided to alter it as a smaller code exercise.

This is an implementation of an accordion menu in Objective-C by customizing UITableView and its delegate and data source protocols.

User will be able to build the menu using either normal UITableViewCell or customized cell nibs.

Transition between the selected contents of the menu will be animated based on choices in UITableViewRowAnimation or users can define their own.

Concordance for Shakespear's Richard III

This is the first group Java project ever !!

This basically show you a GUI where you can enter a keyword or key string to search in the context of Shakespear's Richard III Act I Scene 1 - 4 only.

This comes with auto-search and highlighter features where you can just search a word and you can view all matched strings in each of the four scenes.

Just run the .jar file and it should do it, alone.
